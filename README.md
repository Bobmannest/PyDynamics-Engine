This is a physics engine that uses Python object oriented programming to simulate real-world physics concepts using objects which can freely move around and collide 
with eachother in real time. 

## Features:
- The mouse can be used to interact with the physics objects by drawing them in towards the mouse which will help to visually display
  the range of movements and collisions that physics objects can have.
- Physics objects use random number generation to spawn with varying sizes and colours.
- Physics objects have masses which depend on their radius and gravity will affect them accordingly with their individual masses.
- Physics objects will detect collisions with other Physics objects in real time

## Build Instructions
1) Use Python (version 3.13 and above)
2) Download Python Pygame (version 2.6.1 and above)
3) Run main.py
